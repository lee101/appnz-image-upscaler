# Third-party software and models

- [Real-ESRGAN](https://github.com/xinntao/Real-ESRGAN), BSD-3-Clause, powers
  image restoration and super-resolution.
- [GFPGAN](https://github.com/TencentARC/GFPGAN), Apache-2.0, provides optional
  face restoration.
- Real-ESRGAN and GFPGAN release weights are downloaded from their official
  GitHub releases during the image build. Keep their notices when
  redistributing the resulting image.
- [Cog](https://github.com/replicate/cog), Apache-2.0, provides the container
  build and prediction runtime.
